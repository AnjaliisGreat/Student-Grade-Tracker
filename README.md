# Student Grade Tracker

## Project Description
A simple Java console application that helps students and teachers input marks for different subjects, calculate the average score, and determine pass/fail status for each subject and overall. This project demonstrates basic Java programming concepts such as arrays, methods, and input handling.

## Problem Statement and Objectives
Students and teachers require an easy tool to track and evaluate academic performance quickly. The objectives are:
- Input marks for multiple subjects.
- Calculate average marks.
- Display pass/fail results per subject and overall.

## Technologies Used
- Java (JDK 8 or above)
- Console input/output via Scanner class
